# ■ 国際情勢マップアプリ 引き継ぎ書（矢印UI改善フェーズ）

---

# ■ 現在のフェーズ

フェーズ5：表示改善フェーズ

目的：
地図上の視認性を高め、「直感的に理解できるUI」にする

---

# ■ 実装済み機能（重要）

## ① 地図表示

* React + Leaflet で世界地図表示
* GeoJSONから国データ読み込み

## ② 国クリック

* 国クリックで選択状態になる
* 右パネルにニュース表示

## ③ データ連携

* Supabaseからイベント取得
* `country_iso3` で紐付け

## ④ 関係国矢印（現行実装）

* 選択イベントの
  「対象国 → 関係国」
  に矢印表示
* MapView.jsx で描画

---

# ■ 現在の矢印仕様（最新版）

## ✔ 描画方法

* Polylineは完全廃止
* Polygonのみで描画（帯状）

## ✔ 特徴

* 曲線（ベジェ風）
* 太さあり（リボン型）
* 先端まで一体化

## ✔ 使用関数

* buildFilledArrowPolygon()
* getRibbonOffset()
* movePointBackward()

---

# ■ 現在の問題（重要）

## ① 近距離で崩れる

* 矢印が太すぎる
* 先端が潰れる
* 曲がりが弱く不自然

## ② 視認性の課題

* 色が薄いと見えない
* 黒すぎるとUIが重い

---

# ■ 対応済み改善

## ✔ 改善①：色調整

推奨値：

```js
fillColor: "#334155"
fillOpacity: 0.88
```

影：

```js
fillColor: "#0f172a"
fillOpacity: 0.18
```

---

## ✔ 改善②：先端崩れ修正

* ポリゴンの点順を修正
* 左右対称になるよう再構成

---

## ✔ 改善③：距離別調整（重要）

### 近距離

* 曲率：強く
* 太さ：細く
* 先端：小さく

### 中距離

* 少し強めカーブ

### 遠距離

* なめらかカーブ

---

# ■ 現在のロジック（超重要）

## 距離分類

```js
const isShortDistance = distance < 8;
const isMiddleDistance = distance < 18;
```

---

## 曲線調整

```js
if (isShortDistance) {
  curveBoost *= 2.0;
}
```

---

## 太さ調整

```js
const widthScale = isShortDistance ? 0.72 : 1;
```
