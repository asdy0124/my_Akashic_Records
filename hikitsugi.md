# 国際情勢マップアプリ 引き継ぎ書

## ■ 現在の状態（2026/03時点）

### ▼ デプロイ状況

* Vercelにて公開済み
* Production URL: https://my-akashic-records.vercel.app
* GitHub連携で自動デプロイ

### ▼ 機能実装状況

* 世界地図（React Leaflet）表示
* 国ごとの色分け（ニュース件数ベース）
* 国クリックで右パネルにニュース表示
* 期間フィルタ（7日 / 30日 / 任意日付）
* カテゴリ分類（外交 / 軍事 / 経済）
* CSVデータ読み込み

### ▼ UI構成

* PC：左 地図 / 右 ニュースパネル
* スマホ：上 地図 / 下 ニュース（レスポンシブ対応済）

### ▼ レイアウト構造（重要）

* `.page` → 全体（flex）
* `.page-header` → タイトル
* `.app` → 地図＋右パネル（grid）
* `.map-wrapper` → Leaflet親（高さ100%必須）

※ 地図は「親要素に高さがないと表示されない」ため、
height管理は絶対に崩さないこと

## ■ 注意点（重要）

### ❗ 地図が消える原因

* 親要素の height がない
* flex / grid の min-height 設定不足

### ❗ レイアウト崩れ原因

* height: 100vh の多重指定
* calc の乱用
